## Packages
framer-motion | Smooth animations for page sections and interactions
react-hook-form | Form state management and validation
@hookform/resolvers | Zod resolver for react-hook-form
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
The application requires RTL (Right-to-Left) layout support for Arabic content.
The primary font "Cairo" will be imported from Google Fonts.
Form submission connects to POST /api/registrations.
File upload visual component is included, but currently sends a placeholder string as no upload endpoint was specified in the schema.
